Aplikasi Kost Hunter

Membangun aplikasi mobile "Kost Hunter" berbasis Flutter untuk memudahkan pengguna dalam mencari dan menemukan informasi hunian kost secara cepat dan efisien melalui tampilan yang intuitif.

Sistem Absensi "Evelampp"

Membangun sistem absensi karyawan berbasis web "Evelampp" dengan memanfaatkan MySQL dan phpMyAdmin sebagai pengelola database untuk mencatat dan memantau data kehadiran secara real-time dan otomatis.

Sistem Manajemen stok "AlbertSport"

Membangun situs web iventaris toko olahraga untuk manajemen stok berbasis database MySQL/phpMyAdmin.

Pendeteksi Curah Hujan dan kelembaban Udara berbasis ESP32

Merancang, merakit sebuah alat untuk mendeteksi hujan dan kelembaban udara digunakan untuk sistem irigasi tanaman cabai.
